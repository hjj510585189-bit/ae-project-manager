
一个专为 macOS 设计的 After Effects 项目文件管理工具，支持中英文切换。


## 功能特性

- 📁 管理本地 .aep文件，支持拖拽添加
- 🖼️ 为项目文件设置封面图片或视频预览
- 🗂️ 自定义分类，灵活组织项目文件
- 🔍 实时搜索，快速定位文件
- 🌐 支持中英文界面切换
- 🚀 双击直接用 After Effects 打开文件

## 下载安装

前往 [Releases](../../releases) 页面下载最新版本：

AE文件管理-1.0.0-arm64.dmg — 推荐，直接安装
AE文件管理-1.0.0-arm64-mac.zip — 免安装版

仅支持 Apple Silicon（M1/M2/M3）Mac

## 打开文件损坏隔离的方法

打开终端（Terminal）。
输入命令：sudo xattr -rd com.apple.quarantine
输入一个空格。
将【应用程序】文件夹中的应用拖入终端窗口，回车。
输入系统密码（输入时不会显示），回车即可。
